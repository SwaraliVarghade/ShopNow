# ShopNow

ShopNow is a React-based shopping application that allows users to browse products, add items to cart, and save products for later.

## Features
- Product listing
- Add to Cart
- Save for Later
- Responsive UI
- React Bootstrap Modal

## Technologies Used
- React.js
- JavaScript
- Bootstrap
- HTML
- CSS

## Installation

1. Clone the repository


git clone https://github.com/SwaraliVarghade/ShopNow.git

2. Navigate to project folder

cd ShopNow

3. Install dependencies

npm install

4. Run the project

npm start
